# ImageGallerySearch

## What tools did I use?

    Java 8 (1.8.0_241)
    Maven (4.0.0)
    SpringBoot (2.3.4.RELEASE)
    

## How to run app?

You should have Java and Maven. After installation, you need to clone this repository:

    https://github.com/EduardVavrynchuk/ImageGallerySearch

## And run:

    mvn spring-boot:run
